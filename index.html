RPN Calculator

How to use RPN calculator

The calcultor performs a function on the last two elemenst of the stack.
For example, to add 50 and 30:

Type 50 -> Press Enter -> Type 30 -> Press +

The 'C' button clears the last element of the stack.

The 'AC'' button clears all the stack.

The 'M+','M-','MR','MC', keys adds, subtracts, display and clear the Memory.

The SWAP key swaps the last two elements where as the POP key removes the last element and shift the stack down.
